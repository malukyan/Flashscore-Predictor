# ⚽ Flashscore Predictor v1.0 Professional

**Профессиональный инструмент для AI-анализа футбольных матчей и прогнозирования тотала голов (ТБ/ТМ 1.5) с доверительным интервалом 95%.**

---

## ✨ Ключевые возможности

- 📊 **Точные прогнозы**: вероятности ТБ 1.5, 1X2, обе забьют (BTTS), азиатские форы.
- 🤖 **AI-анализ**: автоматический расчёт ожидаемых голов с учётом формы команд, мотивации, личных встреч и xG (expected goals).
- 💰 **Value Betting**: поиск перевеса над линией букмекера + оптимальный размер ставки по формуле Келли (ограничение 25% от банка).
- ⏱️ **Детализация по таймам**: отдельные прогнозы на 1-й и 2-й тайм (ТБ/ТМ 0.5, 1.0, 1.5).
- 📥 **Экспорт результатов**: в Excel (.xlsx) и PDF с полной статистикой и графиками.
- 🔄 **Live-обновление**: автоматическое обновление счёта для матчей в статусе «Live».
- 🔔 **Уведомления**: напоминание о начале матча с высокой вероятностью ТБ 1.5 (>80%).
- 🛡️ **Гибкая лицензия**: пробный период 3 дня (10 матчей/день), затем активация по ключу.

## 🚀 Установка и первый запуск

1. **Скачайте** `FlashscorePredictor_Setup.exe
` из раздела [Releases](https://github.com/MaLukyan/Flashscore-Predictor/releases/latest).
2. **Запустите** файл от имени администратора (правый клик → «Запуск от имени администратора»).
3. При первом запуске появится **интерактивный тур**. Следуйте подсказкам.
4. Нажмите **«🔄 Обновить матчи»** для загрузки расписания на выбранную дату.
5. Выберите любой матч в таблице слева — справа появится **детальный AI-прогноз**.
6. Используйте фильтры (по лиге, статусу, вероятности) и экспорт в Excel/PDF.

> 💡 Все настройки сохраняются. При обновлении версии просто заменит `.exe` файл — ваши данные останутся нетронутыми.

---

## 🔄 Обновление программы

Программа **автоматически проверяет** новые версии при каждом запуске через API.

- Если вышла новая версия, появится окно: `Доступна новая версия! Перейти на страницу загрузки?`
- Нажмите **«Да»** → откроется страница релиза в браузере.
- Скачайте новый `.exe`, закройте программу, заменит файл.
- Готово! Лицензия, настройки и база данных сохранятся.

## 🔑 Лицензия и пробный период

- 🆓 **Trial-режим**: 3 дня полного доступа. **Лимит**: 10 матчей в день.
- 💎 **Полная версия**: безлимитный анализ, экспорт без ограничений, приоритетная техническая поддержка.
- 📝 **Активация**:
  1. Откройте вкладку **«⚙️ Настройки»**.
  2. Скопируйте **ID устройства** с помощью кнопки «📋 Копировать».
  3. Приобретите лицензионный ключ на официальном сайте [zetabot.ru](https://zetabot.ru).
  4. Вставьте ключ в поле «Лицензионный ключ» и нажмите **«🔑 Активировать»**.
  5. Перезапустите программу.
- 🔁 Лицензия привязана к одному ПК. Для переноса на другое устройство обратитесь в поддержку.
